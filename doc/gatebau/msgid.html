Version 4.2 of 01 May 1997, supersedes the MSGID.DOC version 4.1.

This document describes the conversion between FIDO ^AMSGID/^AREPLY and Internet RFC mail/news Message-ID/References. It is a recommendation for a common handling of message ID passing for FIDONET-Internet gateways.
